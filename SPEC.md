## 프로젝트 사양서 (SPEC)

- **1. 프로젝트 개요**: PHP 웹하드 프로그램 개발
- **2. 프로그램 언어**: PHP + Pure HTML + Vanilla JS
- **3. 참조 프로그램**: filegator
- **4. 목표 구성**: 참조 프로그램(filegator)과 동일한 구성을 가진 웹하드 프로그램 제작
- **5. 주요 기능 (공통)**:
  - 회원관리
  - 로그인
  - 파일 목록 조회
  - 폴더 생성
  - 파일 업로드
- **6. 파일 개별 기능 (파일 목록 화면에서)**:
  - 파일 삭제
  - 파일 다운로드
  - 파일 링크 URL 복사
- **7. 파일 권한 및 링크 정책**:
  - 파일은 **개인파일**, **공개파일**로 구분
  - 파일 링크 URL은 **권한과 관계없이 항상 링크 생성** (단, 실제 접근 권한 정책은 추후 설계에 따름)
- **8. 로그인, 회원등록, 파일리스트는 bootstrap5 를 이용한 디자인 적용**
- **9. 회원, 파일관리는 mysql 이용**:
  - QUERY.sql 파일에 테이블 생성쿼리 적용
- **10. php db접속은 inc 폴더에 include 로 적용**
- **11. html 페이지는 .php 로 생성하고, 디자인에서 공통부분은 include 로 적용**
- **12. 파일리스트에서 다운로드링크 외에 웹뷰링크 적용. 이미지, 영상, 음악 등의 파일은 다운로드 하지 않고 웹브라우저에서 바로 실행이 가능한 링크 제공**
- **13. 이 프로그램에 한해서 파일 업로드 용량 제한을 최대 1GB 까지 늘리도록 수정**
- **14. 파일등록시에 다중 파일 등록이 되게 처리**
